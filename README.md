# CVRP-ACO
Capacitated Vehicle Routing Problem solved with Ant Colony Optimization

## Set up

```
pip install matplotlib
pip install numpy
pip install networkx
```

Download dataset Augerat et al (A) from http://neo.lcc.uma.es/vrp/vrp-instances/capacitated-vrp-instances/
and put it in `data/` dir.

## Run

``` 
python ants.py
```
